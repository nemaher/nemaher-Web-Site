New CSS Features

Web fonts: Web fonts allow you to use a large variety of fonts on your website. This is a lot better then just using the handful of "web-safe" fonts.

This is my pretty webfont!

Gradients: This feature allows you to have smooth gradients on your images without having to rely on the images. This is a new feature that allows CSS to change the gradient without having to import or edit an image.

Linear Gradient

Box Sizing: Box sizing allows you to set the height and width of a box. With the Box Sizing property you can add padding and boarders to your box without it changing the size or shape of the box.

Box Sizing

Boarder-Image: With this property you can set custom boarders around HTML elements. This is a new feature that lets you use a single image that will automatically change size to fit your element.

Multiple Backgrounds: With this feature you can have backgrounds on top of each other compared to just one background. Also with the new transition feature you can also make it appear as if the spaceship is traveling through space.
